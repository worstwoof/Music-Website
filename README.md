## ✨ 功能特性 (Features)

本项目致力于提供一个功能全面、界面美观、交互流畅的现代音乐网站体验。

### 核心功能

* **🔍 智能搜索 (Smart Search)**
    * **即时建议:** 在用户输入时，动态显示相关的歌曲、歌手或专辑建议。
    * **多维搜索:** 支持按歌曲名、歌手、专辑、歌单进行混合搜索。
    * **热门搜索:** 展示当前最热门的搜索词条。
    * **搜索历史:** (本地) 记录用户的最近搜索，方便快速回访。
    * **结果高亮:** 在搜索结果中高亮显示匹配的关键词。

* **❤️ 收藏与喜欢 (Collection & Favorites)**
    * **我喜欢:** 用户可以对任意 *歌曲* 进行“喜欢”（点亮红心）操作，所有喜欢的歌曲会自动汇集到“我喜欢的音乐”列表中。
    * **收藏专辑:** 用户可以收藏整个 *专辑*。
    * **收藏歌单:** 用户可以收藏其他人创建的优质 *歌单*。
    * **关注歌手:** 用户可以 *关注* 喜欢的歌手，以便在歌手页面快速访问。

* **🎤 歌手中心 (Artist Hub)**
    * **歌手主页:** 展示歌手的详细资料、头像、热门歌曲列表、所有专辑及相关的 MV。
    * **歌手分类:** 支持按地区（如华语、欧美、日韩等）和流派对歌手进行筛选和浏览。
    * **字母索引:** 提供 A-Z 字母表快速定位歌手。

### 播放器与体验

* **🎵 全局播放器 (Global Player)**
    * **固定体验:** 播放器固定在页面底部（或侧边），用户在浏览网站其他页面时音乐播放不中断。
    * **完整控制:** 包括播放/暂停、上一首、下一首、音量调节（及静音）。
    * **播放进度:** 可视化的进度条，支持点击和拖拽快进/后退。

* **🎶 播放模式与队列 (Play Modes & Queue)**
    * **三种模式:** 支持列表循环、单曲循环和随机播放。
    * **播放队列:** 可视化的当前播放列表，支持添加、删除歌曲或清空队列。

* **📜 同步歌词 (Lyrics)**
    * **滚动显示:** 在歌曲播放详情页（或播放器展开时），歌词可根据播放进度高亮并自动滚动。
    * **歌词翻译:** (若API支持) 提供外语歌曲的翻译对照。

### 内容发现

* **🏠 首页推荐 (Home Discovery)**
    * **个性化推荐:** (设计占位) 模拟“猜你喜欢”的音乐卡片流。
    * **新歌新碟:** 展示最新发行的歌曲和专辑。
    * **精选歌单:** 推荐高质量的官方或用户歌单。
    * **轮播图 (Banner):** 顶部区域展示热门活动或焦点专辑。

* **🎼 歌单广场 (Playlist Square)**
    * **标签筛选:** 提供多种风格标签（如“流行”、“摇滚”、“电子”、“治愈”）供用户筛选歌单。
    * **歌单详情:** 点击歌单封面进入详情页，展示歌单介绍、标签、歌曲列表和评论。

* **📈 排行榜 (Top Charts)**
    * **多维榜单:** 展示热歌榜、新歌榜、飙升榜、原创榜等多个权威榜单。

* **🎬 MV 视频 (Music Video)**
    * **MV 浏览:** 提供 MV 的发现页面，支持按地区或热度排序。
    * **视频播放:** (设计占位) 专门的 MV 播放页面。

### 界面与交互

* **📱 响应式设计 (Responsive Design)**
    * **全平台适配:** 界面布局自动适应 PC 电脑、平板和智能手机屏幕。
    * **移动端优化:** 针对触摸操作进行交互优化。

* **🎨 主题切换 (Theming)**
    * **深色/浅色模式:** 支持一键切换浅色（Light Mode）与深色（Dark Mode）主题，适应不同环境光。

* **🚀 体验优化**
    * **骨架屏 (Skeleton):** 在数据加载完成前，使用骨架屏占位，提升加载过程中的视觉体验。
    * **过渡动画:** 提供平滑的路由切换动画和优雅的 UI 交互动效。
